## Key Concepts

- **Nyx** = Monad (platform libraries) + apps that compose them
- **Dual workspace**: Cargo (Rust) + pnpm (JavaScript)
- **Multi-app model**: One identity works across apps via app-scoped aliases (privacy isolation)
